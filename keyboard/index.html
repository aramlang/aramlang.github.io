Users must adhere to this order when typing:
  1. Type the base letter (e.g. ܕ )
  2. Then type the Qushaya/Rukakha dots, if any (e.g. ܕ݁ )
  3. Then type Syame, if any (e.g. ܕ݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁̈ )
  4. Then type the vowel, if any (e.g. ܕ݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁݁̈ܰ݁ )
