Rules for Gdhama

The following abbreviations will be used to refer to each
type of Gdhama:
Construct State = Const. Absolute State = Abs.

1- Singular nouns ending in
(not ending in)

All singular nouns ending in (not in) have the same identical form of Gdhama for both the Construct and the Absolute State.
Nouns that are monosyllable are abbreviated by omitting
from the end and giving Zlama Psheeqa to the first letter.
Full Form
Const. & Abs.
  Blood Type,
Class		 Father-in-law Name

Nouns of the form are abbreviated by omitting
from the end and giving Zlama Qashia to the second letter.

Witness		 Friend,
